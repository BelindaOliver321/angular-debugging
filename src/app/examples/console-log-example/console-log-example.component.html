console.log() example


Using console.log() we can get the value of any statement printed out to the console which is seen in your dev tools (F12).

A great first option for figuring out what is going on in your code.

Below find a few guidelines that make working with them easier.


good labeling

When there are many console.log() statements, it's best to label them so that it isn't too confusing.


reference objects

A console.log() of an object is evaluated when it is opened. So if it changes after the log statement, that is the version you will see in the console.
