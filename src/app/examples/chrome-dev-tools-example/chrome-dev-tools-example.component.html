chrome dev tools example


Chrome dev tools (F12) can do step line debugging of your angular application right out of the box.

Below find specific examples of how to do it.


debugger statement

Just put a debugger statement in your code and if the dev tools are open when that line is executed, then it will stop on that line.


break points

You can set breakpoints in the code using the same window you get when you have a debugger statement.
